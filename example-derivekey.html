WebCrypto GOST: Eugene's encrypted Letter

This is a live example. All buttons execute appropriate javascript algorithms with values from the page. For experimentation directly on the page you can change the text or ciphertext, the values ​​of digital signatures, secret keys and passwords, make changes in the javascript algorithms, their settings.

Encryption message for delivery through open channel

Eugene has own generated private key and Tatiana's public key. To encrypt the Letters he needs to create a secret key, which also could easily get Tatiana. He uses derive key function based on asymmetric algorithm for generation KEK (Key Encryption Key) with pseudo random UDM. When Eugene has written Letter to Tatiana, he generates one-time secret CEK (Content Encryption Key) and encrypt message. After Eugene uses generated KEK and UDM for wrapping CEK. Wrapped CEK has (UDM | Encrypted CEK | HMAC) and will send to Tatiana with encrypted message.

                    Encrypt channel procedure
                    
// Get private key from secure repository
gostCrypto.subtle.importKey('raw', gostCoding.Hex.decode(privateEugeneKey.value),
        {name: 'GOSTR3410'}, true, ['deriveKey', 'sign']).then(function(baseKey) {

    // Get peer's public key
    return gostCrypto.subtle.importKey('raw', gostCoding.Hex.decode(publicTatianaKey.value),
            {name: 'GOSTR3410'}, true, ['deriveKey', 'verify']).then(function(pubKey) {
        var UDM = new Uint8Array(8); gostCrypto.getRandomValues(UDM);

        // Use peer's public, own private key and seed for create derive key
        return gostCrypto.subtle.deriveKey({name: 'GOSTR3410', hash: {name: 'GOSTR3411'}, ukm: UDM,
            public: pubKey}, baseKey, {name: 'GOST28147'}, false, ['wrapKey']).then(function(KEK) {
            
            // Generate CEK (Conten Encryption Key)
            return gostCrypto.subtle.generateKey({name: 'GOST28147'}, false, ['encrypt']).then(function(CEK) {

                // Encrypt message by using CEK. Encrypted message will send with wrapped CEK
                return gostCrypto.subtle.encrypt({name: 'GOST28147-CFB'}, CEK,
                        gostCoding.Chars.decode(message.textContent, 'win1251')).then(function(data) {
                    encrypted.textContent = gostCoding.Base64.encode(data);

                    // Wrap CEK by using KEK and UDM pseudo random
                    return gostCrypto.subtle.wrapKey('raw', CEK, KEK, {name: 'GOST28147', ukm: UDM});
                }).then(function(data) {
                    wrappedKey.value = gostCoding.Hex.encode(data);
                    decryptButton.removeAttribute('disabled');
                });
            });
        });
    });
}).catch(function(error) {
    alert(error.message);
});

                

                
Decryption message that delivered through open channel

                

                    
Tatiana unwrap one-time CEK (Content Encryption Key) using own private key and Eugene's public key to derive KEK (Key Encryption Key). Then Tatiana decrypt Letter from Eugene.

                    
                    
                    
                    
                    
                    
                    Now Tatiana can read decrypted Letter from Eugene. Produced and used keys (KEK, UDM and CEK) she burned. She will receive new keys in every packages that contains next messages.
                    

                    
It's not full implementation because message can be changed in the channel. Use Digital signatures as in previous examples for ensure data immutability.

                    Decrypt channel procedure
                    
// Get private key from secluded place
gostCrypto.subtle.importKey('raw', gostCoding.Hex.decode(privateTatianaKey.value),
        {name: 'GOSTR3410'}, true, ['deriveKey', 'sign']).then(function(baseKey) {

    // Get peer's public key
    return gostCrypto.subtle.importKey('raw', gostCoding.Hex.decode(publicEugeneKey.value),
            {name: 'GOSTR3410'}, true, ['deriveKey', 'verify']).then(function(pubKey) {
        
        // Get UDM - first 8 bytes in wrapped CEK
        var wrappedCEK = gostCoding.Hex.decode(wrappedKey.value), UDM = new Uint8Array(8);
        UDM.set(new Uint8Array(wrappedCEK, 0, 8));

        // Use peer's public, own private key and seed for create derive key
        return gostCrypto.subtle.deriveKey({name: 'GOSTR3410', hash: {name: 'GOSTR3411'}, ukm: UDM,
            public: pubKey}, baseKey, {name: 'GOST28147'}, true, ['unwrapKey']).then(function(KEK) {

            // Unwrap CEK using KEK
            return gostCrypto.subtle.unwrapKey('raw', wrappedCEK, KEK, {name: 'GOST28147', ukm: UDM},
                {name: 'GOST28147'}, false, ['decrypt']);
        }).then(function(CEK) {

            // Decrypt message
            return gostCrypto.subtle.decrypt({name: 'GOST28147-CFB'}, CEK,
                    gostCoding.Base64.decode(encrypted.textContent));
        }).then(function(data) {
            decrypted.textContent = gostCoding.Chars.encode(data, 'win1251');
        });
    });
}).catch(function(error) {
    alert(error.message);
});
